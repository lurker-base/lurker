how scoring works

Every new token is analyzed against multiple criteria:

+ Liquidity — Higher liquidity = easier entry/exit (up to 30 pts)
+ Volume — Trading activity indicates interest (up to 25 pts)
+ Holders — Distribution reduces manipulation risk (up to 20 pts)
+ Verification — Verified contract = transparent code (15 pts)
- Red flags — Mint functions, honeypot risk, high tax (penalties)

Score 80+ = Green (low risk) | 60-79 = Orange (medium risk) | Below 60 = Red (high risk)

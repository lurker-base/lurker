## 🔧 Seuils de détection (ajustables)

```python
CONFIG = {
    "min_liquidity": 1000,      # $ minimum pour être détecté
    "min_volume_5m": 500,       # $ volume 5min minimum
    "max_age_minutes": 10080,   # 7 jours max
}
```

Fichier: `scripts/lifecycle_core.py` (lignes 11-17)

```python
THRESHOLDS = {
    "cio_max_age": 10,          # min → CIO
    "watch_max_age": 30,        # min → WATCH
    "hotlist_max_age": 60,      # min → HOTLIST
    "fast_max_age": 1440,       # min → FAST
    "certified_min_age": 1440,  # min → CERTIFIED
    "rug_liq": 0,               # $0 = RUGGED
}
```
